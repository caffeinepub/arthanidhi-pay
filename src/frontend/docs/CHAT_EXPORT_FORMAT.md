# Chat-Based Export Format

This document specifies the exact format for delivering complete repository code via chat when the Export Code button is not available (Basic plan users).

## Purpose

For users on the Basic plan who cannot access the Export Code feature, this format provides a structured, copy-paste-friendly way to receive the complete project codebase directly in the chat interface.

## Format Specification

### File-by-File Listing

Each file is delivered with:
1. **Clear file path** - The exact relative path from the project root
2. **Language identifier** - For syntax highlighting (e.g., `typescript`, `motoko`, `css`, `json`)
3. **Complete file contents** - No abbreviations, placeholders, or omissions
