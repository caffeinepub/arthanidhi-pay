# Local Development Guide

This guide will help you set up and run the ArthaNidhi Pay User Portal locally on your machine.

## Prerequisites

Before you begin, ensure you have the following installed:

- **Node.js** (v18 or higher) - [Download here](https://nodejs.org/)
- **pnpm** (v8 or higher) - Install with: `npm install -g pnpm`
- **Git** - [Download here](https://git-scm.com/)
- **DFX** (Internet Computer SDK, v0.15.0 or higher) - Only required for IC mode
  - Install with: `sh -ci "$(curl -fsSL https://internetcomputer.org/install.sh)"`
